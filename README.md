# Kiru Landing

Kiru Landing is a modern travel and fashion experience built with Vite, React, TanStack Router, and TanStack Start.

## Features

- Travel planning and packing guidance flows
- Outfit and wardrobe intelligence experiences
- Responsive marketing and product pages
- Modern UI components built with Radix and Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local preview in your browser.

## Available Scripts

- `npm run dev` – start the Vite development server
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally
- `npm run lint` – run ESLint

## Project Notes

- The app uses file-based routing via the `src/routes` directory.
- Generated route trees are managed by the TanStack router plugin.
- Environment variables should be kept in local files and not committed.
